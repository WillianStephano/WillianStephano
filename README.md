## Sobre mim

Sou o **Willian Stephano**, tenho 26 anos, sou de São Paulo e iniciei minha jornada em tecnologia de forma autodidata em 2021, estudando **HTML, CSS e JavaScript**.

Atualmente atuo como **Cybersecurity Consultant**, trabalhando com criação de políticas, procedimentos, processos de segurança, análise de evidências, assessments e entrevistas com áreas de negócio.

Sou formado em **Desenvolvimento de Software Multiplataforma pela Fatec Cotia** e tenho experiência anterior em **operações de TI no Bradesco**, onde desenvolvi habilidades em resolução de problemas, trabalho em equipe, adaptação a ambientes corporativos e atuação em rotinas críticas de tecnologia.

Também fui estudante de **Filosofia na USP (FFLCH)**, experiência que fortaleceu minha capacidade analítica, pensamento crítico e criatividade na resolução de problemas complexos.

Hoje, busco evoluir na interseção entre **desenvolvimento de software, segurança da informação, automação, cloud security e AppSec**.

**Tecnologias:** Python, Flask, Gemini API, HTML, CSS e JavaScript

[Ver projeto](https://github.com/WillianStephano/Gerador-de-Provas-IA)

### Bora Trocar

Plataforma web e mobile para troca de produtos entre usuários, com proposta semelhante a um marketplace focado em trocas. O projeto inclui autenticação, filtros por categoria, chat para negociação e estrutura full-stack.

**Tecnologias:** Angular, React Native, Firebase, Spring Boot e Docker

[Ver organização](https://github.com/BoraTrocar)
